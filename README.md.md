# Сайт учителя информатики и математики

Сайт-портфолио Игнатовой Наталии Александровны, участницы конкурса "Учитель года 2026".

## Структура проекта

- `index.html` - Главная страница
- `style.css` - Стили
- `script.js` - JavaScript
- `assets/` - Ресурсы (изображения, шрифты)
- `manifest.json` - PWA манифест

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)
- Font Awesome иконки
- Google Fonts

## Размещение

Сайт готов к размещению на любом хостинге:
1. GitHub Pages
2. Netlify
3. Vercel
4. Любой веб-хостинг
